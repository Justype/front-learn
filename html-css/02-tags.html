HTML 标签

学习基本的HTML标签，如果标题是超链接，链接的是MDN中文文档。

排版标签

标题标签

  • <h1>,<h2>,<h3>,<h4>,<h5>,<h6>（共六种）
  • 根据重要性逐级递减
  • <h1>的语义很重，一般一个网页只有一个，给logo使用

段落标签

<p>，paragraph 段落，就是用来显示段落的

水平线标签

<hr>，HTML5支持不闭合的，也支持闭合的<hr/>

换行标签

<br> 与<hr>相似，HTML5支持不闭合的，也支持闭合的<br/>

div & span

<div> 为块级标签，<span> 为行内标签。

  • 行内标签不能包裹块级标签
  • 块级标签 默认 自动换行

文本格式化标签

  • 加粗，<b>(bold) <strong>
  • 斜体，<i>(italic) <em>(emphasis)
  • 删除，<s> <del>(delete)
  • 下划线，<u>(underline) <ins>(insert)

HTML 推荐使用strong，em等带有语义的标签，就如同HTML的名字一样的，超文本标记语言，就是因为该文本重要，才会加粗

标签属性

<标记名 属性1="属性值" 属性2="属性值">

  • 可以有多个属性
  • 属性有默认值，需要自定义再写
  • 属性值建议用英文双引号包裹

图像标签

<img src="图片地址" alt="代替文本" height="" weight="" border="" title="鼠标hover时显示的文本">

  • 高度，宽度设置一个的话，按比例自动缩放
  • 图片标签是单标签，与hr、br相似
  • 行内标签，不会自动换行

链接标签

<a href="目标地址" target="打开网页的方式">

Microsoft

锚点定位

base标签

在<head>内添加<base target="打开网页的方式"> // 单标签

设置当前网页默认标签的打开方式

HTML 特殊字符

&特殊的代码;

  • 空格：nbsp
  • 小于号：lt

注释标签

<!-- 要注释的内容 -->

注释不会在网页内显示，方便代码维护

相对路径

网页中一般使用相对路径，如果工作区在主机中转换了位置，绝对路径会改变，通过之前的绝对路径，找不到原来的文件，所以要使用相对路径。

  • 同一级 直接写文件名 或 ./文件名
  • 上一级文件夹 ../
  • 下一级文件夹 文件夹名/
