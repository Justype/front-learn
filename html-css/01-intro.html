Web标准

  • Behaviour: ECMAScript,DOM,BOM

HTML

Hyper Text Markup Language，即超文本标记语言

举例： 

  • <strong>加粗</strong>
  • 加粗

骨架

<html>
    <head>
        <title></title>
    </head>
    <body>
        content
    </body>
</html>

特点

  • html标签可以嵌套，就像是head在html标签内一样

字符集

  • 由于拥有不同的编码，所以显示情况会有不同，目前一般使用utf-8，拥有几乎所有的字符，全球通用
  • 网页的编码应与文件的编码相同，如果用记事本，注意保存格式
